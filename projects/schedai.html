SchedAI

  • Tech Stack: Python, Machine Learning, Django, HTML, CSS, JavaScript
  • Project Type: AI-Powered Scheduling System
  • Github URL: Project Link

SchedAI is an intelligent scheduling assistant that uses machine learning to help users optimize their daily schedules. The app analyzes user preferences, priorities, and time constraints to suggest optimal scheduling solutions.
