ZamboFare: Zamboanga Tricycla Fare Calculator

  • Tech Stack: React Native, Node.js, Express, MongoDB, Firebase, HTML, CSS, JavaScript
  • Project Type: Mobile Transportation & Fare Management System
  • Github URL: Project Link

ZamboFare App is a comprehensive transportation and fare management system designed for Zamboanga City. The application provides real-time fare calculation, route planning, payment processing, and transportation tracking to enhance the commuting experience for residents and visitors.
