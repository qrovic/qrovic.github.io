Meet N' Eat Kiosk Ordering System

  • Tech Stack: React, Node.js, Express, MongoDB, Socket.io, HTML, CSS, JavaScript
  • Project Type: Food Court Kiosk & Ordering System
  • Github URL: Project Link

MeetNeat is a comprehensive food court kiosk and ordering system that streamlines the food ordering process. The platform features an intuitive touch-screen interface for customers to browse menus, place orders, and make payments, while providing restaurant staff with real-time order management capabilities.
