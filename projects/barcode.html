Barcode HR & Payroll System

  • Tech Stack: HTML, CSS, JavaScript, PHP, MySQL
  • Project Type: RFID Based Attendance and Payroll System
  • Github URL: Project Link

Barcode HR & Payroll System is a comprehensive system that enables users to manage their HR and payroll information. The system features a barcode scanner for attendance and payroll management, as well as a database for storing employee information and payroll data.
